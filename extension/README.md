# YouTube Analytics Tracker

A Chrome extension that tracks detailed statistics for your YouTube subscriptions:
- Watch duration for each video
- Like and comment activity tracking
- Per-channel analytics
- Data export in CSV/JSON formats

## Features

- **Video Tracking**: Records when you watch videos, how long you watch them, and which actions you take
- **Channel Statistics**: Aggregates data by channel to show which channels you engage with the most
- **Modern Dashboard**: View your YouTube habits in an intuitive, visually appealing dashboard
- **Data Export**: Export your statistics for further analysis

## Installation (Development Mode)

1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" using the toggle in the top-right corner
4. Click "Load unpacked" and select the extension directory
5. The extension is now installed and will start tracking your YouTube activity

## Usage

1. Browse YouTube as normal
2. Click the extension icon to view your statistics
3. Use the dashboard to analyze your viewing habits
4. Export data using the CSV or JSON buttons

## Privacy

This extension stores all data locally on your device and does not transmit any information to external servers. Your YouTube viewing data remains private.

## Storage

All statistics are stored in Chrome's local storage and are persistent until you clear your browser data or uninstall the extension.
