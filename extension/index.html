YouTube Analytics Tracker

Track detailed statistics for your YouTube subscriptions

Features

  • Track watch duration for each video
  • Record like and comment activity
  • View analytics by channel
  • Visualize your YouTube engagement with charts
  • Export data as CSV or JSON for further analysis

Installation Instructions

  1. Download the extension files to your computer
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable "Developer mode" using the toggle in the top-right corner
  4. Click "Load unpacked" and select the extension directory
  5. The extension is now installed and will begin tracking your YouTube activity

How to Use

  1. Browse YouTube as normal - the extension will track your activity automatically
  2. Click the extension icon in your browser toolbar to open the dashboard
  3. View statistics by channel or across all channels
  4. Use the filter options to analyze specific time periods
  5. Export your data using the CSV or JSON buttons for further analysis

Privacy

This extension stores all data locally on your device and does not transmit any information to external servers. Your YouTube viewing data remains private and is stored in Chrome's local storage.

You can clear all stored data at any time by right-clicking the extension icon, selecting "Options", and clicking "Reset Data".
